## 🔧 故障排除

### 问题 1: 看不到更新的效果
**解决方案**：强制刷新浏览器
- 按 `Ctrl + Shift + R` (Windows/Linux)
- 按 `Cmd + Shift + R` (Mac)
- 或使用无痕模式：`Ctrl + Shift + N` / `Cmd + Shift + N`

### 问题 2: 布局错乱
**解决方案**：检查浏览器控制台
- 按 `F12` 打开开发者工具
- 查看 Console 标签是否有错误
- 查看 Network 标签确认所有 CSS 文件加载成功（状态码 200）

### 问题 3: 输入框不显示
**解决方案**：检查 JavaScript
- 打开浏览器控制台（F12）
- 输入 `document.getElementById('userInput')`
- 如果不为 null，说明元素存在，可能是样式问题

### 问题 4: WebSocket 未连接
**解决方案**：
1. 确认后端服务器正在运行
2. 查看服务器日志是否有 WebSocket 连接记录
3. 使用诊断工具测试：`http://localhost:8000/static/diagnose.html`
